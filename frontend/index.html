📥 Descargas

Todo lo que necesitás para conectarte al servidor

🎮

Cliente Lineage 2 H5

Cliente completo del juego. Necesario si no tenés instalado Lineage 2 Interlude / High Five.

~5.5 GB Cliente completo
⬇ Descargar Cliente
⚙️

System (Parche)

Carpeta System actualizada del servidor. Reemplazá la carpeta System dentro de tu instalación de L2.

~19 MB Parche
⬇ Descargar System
🚀

Launcher

Launcher oficial del servidor. Se encarga de mantener el cliente actualizado automáticamente antes de iniciar el juego.

~15 MB Launcher
⬇ Descargar Launcher

📋 Guía rápida de instalación

1
Descargá el Cliente si no tenés L2 H5 instalado. Si ya lo tenés, podés saltear este paso.
2
Descargá e instalá el System (parche). Descomprimí y reemplazá la carpeta system dentro de tu directorio de Lineage 2.
3
Descargá el Launcher y ejecutalo. Se conectará automáticamente al servidor y mantendrá tu cliente al día.
4
¡Creá tu cuenta desde este sitio y comenzá a jugar! 🗡️
